# go-lambda-dynamo

a basic project template for writing lambda functions (+ dynamo bindings) with go 

steps:

1. clone repo
```zsh
git clone git@github.com:waymobetta/go-lambda-dynamo.git
```

2. create lambda function within AWS

3. run build script
```zsh
chmod +x /scripts/proc.zsh
./scripts/proc.zsh
```
